# ``CacheKit``

<!--用一句话来描述我的框架使用来做什么的-->
Cache is a lightweight thread safe disk caching tool for Apple platform, commonly used to store data returned from network requests.

## Overview
<!--添加更详尽的信息来描述组件的主要功能-->

CacheKit provides models and utilities for creating, tracking, and removing for caches. The framework provides structures to model an individual ``DiskCache``.
